10 Rules từ thấp đến cao (specificity):

# CSS Rule Specificity Color nếu chỉ rule này
1 * { color: dimgray; } (0,0,0) dimgray
2 p { color: slategray; } (0,0,1) slategray
3 div p { color: steelblue; } (0,0,2) steelblue
4 .text { color: teal; } (0,1,0) teal
5 .highlight { color: darkcyan; } (0,1,0) darkcyan
6 p.text { color: mediumblue; } (0,1,1) mediumblue
7 p.text.highlight { color: darkviolet; } (0,2,1) darkviolet
8 #demo { color: darkred; } (1,0,0) darkred
9 #demo.text { color: crimson; } (1,1,0) crimson
10 ★ #demo.text.highlight { color: hotpink; } (1,2,1) ← THẮNG hotpink

Giải thích kết quả:

Element <p id="demo" class="text highlight"> có màu hotpink vì rule số 10 #demo.text.highlight có specificity (1,2,1) cao nhất trong 10 rules.


Nếu đổi thứ tự rules trong file CSS:

Chỉ khi 2 rules có cùng specificity thì thứ tự mới quan trọng (rule sau thắng). Nhưng trong bài này 10 rules có specificity khác nhau hoàn toàn → đổi thứ tự KHÔNG thay đổi kết quả. Specificity quyết định, không phải vị trí.
