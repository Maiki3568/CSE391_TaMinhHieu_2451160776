Câu C2 — Cascade Puzzle

Dự đoán kết quả (trước khi chạy):

Phần tử font-size color Lý do
"Sản phẩm A" (h2.title.highlight) 20px green .highlight có !important → thắng #featured .title
"Mô tả sản phẩm" (p trong #featured) 14px blue color: inherit → lấy từ .card { color: blue }
"Sản phẩm B" (h2.title) 20px blue Kế thừa từ .card { color: blue }
"Mô tả sản phẩm B" (p.highlight) 14px green .highlight có !important → thắng tất cả

Kiểm chứng thực tế:

Sản phẩm A

Mô tả sản phẩm

Sản phẩm B

Mô tả sản phẩm B

Chú thích:

  • "Sản phẩm A": màu green — .highlight { color: green !important; } thắng #featured .title { color: red; } dù ID selector có specificity cao hơn, vì !important override tất cả.
  • "Mô tả sản phẩm": màu blue — .card p { color: inherit; } → inherit từ cha .card { color: blue; }.
  • "Sản phẩm B": màu blue — h2 kế thừa từ .card { color: blue; }. font-size = 20px từ .card .title.
  • "Mô tả sản phẩm B": màu green — .highlight { color: green !important; } thắng.
